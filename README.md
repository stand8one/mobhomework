# 📚 小学生作业成长助手

> 让孩子看见自己的进度、不断攻克题目，逐渐建立「我能行」的自信。

## 系统概述

本系统由三个组件组成，配合完成对孩子作业的实时感知和进度管理：

| 组件 | 形态 | 核心职责 |
|------|------|----------|
| **Android App** | 手机架在高拍仪/支架上 | 作业拍照录入、定时双模式采集（照片+视频）、番茄钟 |
| **AI Agent** | 云端服务 | 题目解析（题级颗粒度）、进度追踪、效率分析、异常检测 |
| **Web App** | 浏览器端 | 家长省心看板、作业报告、设置管理 |

## 核心理念

- **成长助手，而非监控工具** — 督促不是目的，让孩子建立自信才是目的
- **进度赛跑** — 孩子看到「实际进度 vs 预估计划」，产生超越目标的动力
- **家长省心** — 默认不打扰，仅在进度明显落后时推送通知
